Jekyll Post Card

Beautiful, responsive post cards for your Jekyll site. Embed internal posts or external articles with a simple Liquid tag.

External Article

Embed articles from any website using the full URL.

{% post_card https://r0x0d.com/posts/packaging-for-fedora-fuzzytail-watching-logs-on-terminal %}
{% raw %}{% post_card https://r0x0d.com/posts/packaging-for-fedora-fuzzytail-watching-logs-on-terminal %}{% endraw %}

Card Variants

Compact Variant

{% post_card https://r0x0d.com/posts/packaging-for-fedora-fuzzytail-watching-logs-on-terminal variant:compact %}

Hide Image

{% post_card https://r0x0d.com/posts/packaging-for-fedora-fuzzytail-watching-logs-on-terminal hide_image:true %}
Note: Card variants can be applied using parameters: {% raw %}{% post_card /my-post variant:compact %}{% endraw %} or {% raw %}{% post_card /my-post hide_image:true %}{% endraw %}

Perfect for showcasing a series of related articles or a "You might also like" section.

Tip: Wrap multiple {% raw %}{% post_card %}{% endraw %} tags in a container with display: grid to create multi-column layouts.

Installation

# Add to your Gemfile
gem 'jekyll-post-card'

# Or install directly
gem install jekyll-post-card

# Add to _config.yml
plugins:
  - jekyll-post-card
